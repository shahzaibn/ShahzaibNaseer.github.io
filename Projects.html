Space Shooter (Game)

This project is an interactive survival game coded in Python using Pygame. Navigate your spacecraft and shoot down enemies in a race to survive for the longest.

Line Following Robot

This project is a simple yet innovative line-following robot designed using analog circuitry and discrete logic components. Utilizing infrared sensors and operational amplifiers, the robot follows predefined paths by detecting and tracking black lines on a contrasting surface. It offers an engaging introduction to robotics without the need for microcontrollers or programming.

Library Management System

This project is a comprehensive library management system developed using Python and SQL. It allows librarians to efficiently manage library resources, including books, patrons, and borrowing records. Users can search for books, check their availability, and borrow or return items seamlessly.
